## üìå About the Project

EventHive is a mobile and web-based application designed to enhance college community integration by simplifying event coordination, venue booking, and approval workflows. It fosters collaboration among students, campus leads, and faculty while incorporating gamification to encourage engagement.

## ‚ú® Core Features

*   **Role-Based Access**:

    *   **Students:** Join communities, register for events, provide feedback, view announcements, manage their profiles, and log in or register.
    *   **Campus Leads:** Announce events, award badges, manage internal/external events, add/update venue bookings, and approve/reject requests.
    *   **Faculty in Charge:** Approve/reject event publications, manage the event calendar, analyze feedback.
    *   **Super-Admin:** Manages all users and performs maintenance of mobile app and webpage.
*   **Community Dashboard**: View events, teammates, feedback, and membership forums.  Carousel view for community photos and event highlights.
*   **Event Management**: Create, edit, and manage events. Venue booking with real-time availability. Event feedback analysis and reporting.
*   **Approval System**: Seamless workflow for event, venue, and membership approvals.
*   **Gamification**: Earn badges (Bronze, Silver, Gold, Platinum) based on contributions and roles.
*   **Internship Opportunities**: Display relevant internships with skill requirements and registration links.
